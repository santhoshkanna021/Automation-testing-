# 🏦 Banking App Test Automation Suite

## 🔍 What It Does
- Logs into TestFire banking app
- Transfers funds between accounts
- Validates successful transaction

## 🧪 Tools Used
- Python 3.13
- Selenium
- Pytest
- Pytest-HTML (Reporting)

## ▶️ How to Run
```bash
pytest --html=reports/report.html --self-contained-html
